Welcome to **Mystic Blog** — a space where I write about code, design, and everything in between.

## Why build a blog?

A blog is more than just a place to write. It's a:

- **Learning tool** — explaining concepts solidifies understanding
- **Portfolio** — shows what you can build
- **Archive** — tracks your thoughts over time

## Tech Stack

This blog is built with:

- **Next.js 14** — App Router, Server Components
- **Supabase** — Auth, Database, Realtime
- **Three.js** — Interactive 3D backgrounds
- **Tailwind CSS** — Utility-first styling
- **MDX** — Write content in Markdown

## Code Example

Here's a simple Supabase query:

```typescript
import { createClient } from "@/lib/supabase/server"

async function getComments(postSlug: string) {
  const supabase = await createClient()
  const { data } = await supabase
    .from("comments")
    .select("*")
    .eq("post_slug", postSlug)
    .order("created_at", { ascending: false })

  return data
}
```
